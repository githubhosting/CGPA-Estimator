# CGPA Estimator React App

This is a simple React app that estimates your CGPA based on your grades and credit hours. It is a simple app that can be used to estimate your CGPA for a semester or a year.

## Features

- Add courses
- Add grades
- Add credit hours
- Calculate CGPA

## Technologies

- React
- React Hooks
- React Context API
- React Router
- Material UI
